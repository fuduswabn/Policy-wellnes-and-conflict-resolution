## Branding Customization

### Customizing Your White-Label Identity

1. **Go to Branding Tab**
2. **Upload Logo** (JPG, PNG, SVG)
- Recommended size: 200x200px
- Max file size: 5MB
3. **Set Brand Colors**
- **Primary Color**: Main action buttons & highlights
- **Secondary Color**: Alternative CTAs & links
- **Accent Color**: Warnings, notifications, secondary highlights
4. **Company Name**: Displays across the entire platform
5. **Support Info**: Email, phone (shown to end users)
6. **Website URL**: Optional link in footer

### Color Best Practices

- Use high contrast colors (primary vs background)
- Test on mobile screens
- Avoid too many colors (stick to 3 main colors)
- Ensure accessibility (WCAG AA standards)

**Color Picker Tool**:
- Use hex codes: #RRGGBB
- Or use online color pickers: https://www.color-hex.com/

✓ All screens & buttons

## End User Support

### Your Support Responsibilities

**Tier 1 (You - Reseller)**
- Account access issues
- Password resets
- Billing inquiries
- Feature explanations
- General troubleshooting

**Tier 2 (Us - Provider)**
- Technical bugs
- API errors
- Infrastructure issues
- Data recovery
- Advanced troubleshooting

### Support Escalation

If an end user has a technical issue:

1. **Gather Information**
- Error messages (screenshot)
- Steps to reproduce
- Browser/device type
- User's company & role

2. **Contact Support**
- Email: support@[provider-domain].com
- Include company name & user email
- Attach screenshots/logs
- Mark as urgent if needed

3. **What We'll Do**
- Investigate within SLA (24-48 hours typically)
- Provide technical explanation
- Implement fix if needed
- Test before rollout

### Common Issues & Solutions

**"Forgot password"**
- Click "Forgot Password" on login
- Enter email
- Follow reset link
- If not received: Check spam folder

**"Can't create account"**
- Verify invite code is active (not expired)
- Check email isn't already registered
- Ensure company subscription is active

**"Quiz isn't showing up"**
- Check if quiz is published
- Verify user's group is targeted
- Check quiz schedule/date

**"Data isn't syncing"**
- Refresh browser/app
- Clear cache
- Check internet connection
- Restart app

## FAQ

### General Questions

**Q: Can I modify the platform's code?**
A: No. You can only customize branding (colors, logo, company name). The platform code remains ours.

**Q: Can I reverse-engineer the platform?**
A: No. The agreement prohibits reverse engineering, decompiling, or copying source code. Violations result in immediate termination.

**Q: Can I sell the platform to others as my own?**
A: No. You must maintain proper attribution and cannot sublicense. This is a white-label partnership, not a license transfer.

**Q: What if I go out of business?**
A: Your data will be available for 30 days after termination. Contact support for data export options. Extended backups available for $99/month.

**Q: Can I use a custom domain?**
A: Yes (Professional & Enterprise plans). We can set up DNS records so end users see your domain in URLs.
